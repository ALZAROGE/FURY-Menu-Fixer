# Windows 11 Classic Context Menu Toggle 🛠️

أداة بسيطة وخفيفة تمكنك من التبديل بين قائمة "كلك يمين" الخاصة بويندوز 11 والقائمة الكلاسيكية الخاصة بويندوز 10 بضغطة زر واحدة.

---

### ✨ المميزات (Features)
* **سهولة مطلقة:** لا تحتاج لتثبيت، مجرد ملف `Batch` صغير.
* **آمنة 100%:** تعتمد على أوامر الويندوز الرسمية لتعديل الريجستري.
* **تطبيق فوري:** تقوم الأداة بإعادة تشغيل "مستكشف الملفات" تلقائياً لتفعيل التغييرات.

---

### 🚀 طريقة التشغيل (How to use)
1. قم بتحميل ملف `ClassicMenu.bat`.
2. اضغط بيمين الماوس على الملف واختر **تشغيل كمسؤول (Run as administrator)**.
3. اختر رقم العملية التي تريدها (تفعيل أو إلغاء) ثم اضغط **Enter**.
